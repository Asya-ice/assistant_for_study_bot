# Бот-ассистент Твой помощник в учебе

Телеграм бот, который поможет тебе в учебе.


## Описание

Бот помогает выучить новые слова или определения, понятия,
поможет тебе подготовиться к экзаменам, сессии.
Всё это в формате "вопрос-ответ". Бот будет напоминать о занятии,
так что ты будешь развивать свою память каждый день.

## Инструкции по использованию
- Начало работы:
  - Найди бота в Telegram: 
  - Запусти бота командой /start.
  - Представься и командой /add_card создай свою первую карточку!
- Запоминай новое проще:
  - Каждый день ты будешь повторять нужный для тебя материал и запоминать его.
  - Ты также будешь следить за своей статистикой. 
- Создавай каталоги:
  - Если тебе надо учить несколько тем одновременно, то можешь объединять карточки по группам. 
  - Перед началом занятия бот будет спрашивать какую тему ты хочешь повторить.
- Выбирай удобное время для повторений:
  - Просто пиши боту в любое время, чтобы повторить материал. 
  - Создавай напоминая и повторяй в одно и то же время каждый день. Так ты лучше все запомнишь.
